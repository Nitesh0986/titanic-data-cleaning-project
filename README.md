# Titanic Data Cleaning Project

## Overview

This project focuses on data cleaning and preprocessing of the Titanic dataset. The objective is to prepare raw data for machine learning by handling missing values, encoding categorical variables, and performing basic data analysis.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-Learn

## Tasks Performed

1. Loaded the Titanic dataset using Pandas.
2. Explored the dataset using:

   * head()
   * tail()
   * info()
   * describe()
3. Identified missing values.
4. Handled missing values using median and mode imputation.
5. Encoded categorical columns such as Sex and Embarked.
6. Visualized Age distribution using Matplotlib.
7. Saved the cleaned dataset as a new CSV file.
